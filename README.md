# FontoryApp

손글씨 기반 폰트 생성 및 적용을 실험해 보는 React Native 앱입니다.

## Build with android studio 

```sh
# Metro 번들러 서버 실행 
npx react-native start --reset-cache

# Andriod Studio로 빌드 
npx react-native run-android
```
